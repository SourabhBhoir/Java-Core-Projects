
# Hospital Management System

A simple Java-based Hospital Management System built using Object-Oriented Programming (OOP) concepts and Java Collections Framework.

This project is designed for beginners to practice core Java concepts such as classes, objects, encapsulation, constructors, ArrayList, loops, switch-case, and user input handling using Scanner.

---

## Features

- Add Patient
- View Patients
- View Doctors
- Delete Patient
- Menu Driven Console Application
- Dynamic Data Storage using ArrayList
- Formatted Table Output

---

## Technologies Used

- OOP Concepts
